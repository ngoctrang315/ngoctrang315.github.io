Tạo bảng với Glue

Trong bài lab này, chúng ta sẽ sử dụng AWS Glue Crawlers để quét dữ liệu bán hàng và tạo một bảng mới trong AWS Glue Data Catalog. Sau đó, chúng ta sẽ sử dụng Amazon Athena để truy vấn bảng đó.

Bước 1: Truy Cập AWS Glue Console

  1. Mở AWS Management Console.
  2. Gõ Glue vào thanh tìm kiếm và chọn AWS Glue từ kết quả tìm kiếm để mở AWS Glue Console.

Bước 2: Tạo Crawler

  1. Trong AWS Glue Console, chọn Crawlers từ menu bên trái.
  2. Nhấp vào nút Create Crawler.

Bước 3: Đặt Tên và Cấu Hình Data Source

  1. Nhập tên Athena Sales cho crawler và nhấn NEXT.

  2. Khi được hỏi “Is your data already mapped to Glue tables?”, chọn Not yet.

  3. Nhấn Add a Data Source.

  4. Trong mục Data Source, chọn S3.

  5. Nhấn vào Browse S3, rồi làm theo các bước:

    • Chọn bucket có tên bắt đầu bằng athena-workshop-.
    • Vào thư mục Basics, sau đó chọn thư mục parquet.
    • Nhấn vào ô tròn bên trái thư mục sales.
    • Nhấn Choose.

  6. Giữ nguyên các cài đặt còn lại và nhấn Add an S3 Data source.

Bước 4: Tạo IAM Role

  1. Nhấn Next.
  2. Click the Create new IAM Role button Enter AWSGlueServiceRole-salescrawler as the role name and click the Create button
  3. Click the Next button
  4. Trong phần Set output and scheduling, làm theo các bước sau:
    • Target Database: Chọn default.
    • Table Name Prefix (tuỳ chọn): Nhập athena_glue_.
  5. Nhấn Next.

Bước 5: Xác Nhận và Tạo Crawler

  1. Trên màn hình Review and Create, nhấn Create Crawler.
  2. Sau khi crawler được tạo, nhấn Run crawler. Quá trình này có thể mất từ 2-4 phút để hoàn thành.
  3. Sau khi crawler hoàn thành, bạn sẽ thấy một bảng mới được thêm vào.

Truy Vấn Bảng Mới Với Athena

Sau khi tạo bảng thành công, chúng ta sẽ quay lại Athena Console và chạy một truy vấn.

Bước 6: Truy Cập Athena Console

  1. Mở AWS Management Console và gõ Athena vào thanh tìm kiếm.
  2. Chọn Athena từ kết quả tìm kiếm để mở Athena Console.

Bước 7: Truy Vấn Bảng

  1. Trong thanh bên trái của Query Editor, bạn sẽ thấy bảng athena_glue_sales trong danh sách các bảng.
  2. Nhấn vào biểu tượng ba chấm (ellipsis) bên phải tên bảng và chọn Preview Table.
  3. Truy vấn sẽ chạy và trả về dữ liệu bán hàng.

Chúc mừng! Bạn đã thành công trong việc tạo bảng mới bằng AWS Glue Crawler và truy vấn bảng đó bằng Amazon Athena.
